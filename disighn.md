# ИИSMM - Design System ("disighn.md")

This document specifies the UI styling guidelines, color markers, layout geometry, animation parameters, and visual principles implemented across the **ИИSMM** application.

## 1. Цветовая палитра и фоновые градиенты (Pastel Gradient)
*   **Фоновый градиент:** Пастельное смешение небесно-голубого, нежно-оранжевого и пудрово-розового цветов.
*   **Реализация:** CSS-анимация `animate-gradient-flow` с циклическим изменением положения слоев градиента, создающая ощущение легких облаков, как на референсных скриншотах.
*   **Классы фона:** `bg-gradient-to-tr from-sky-100 via-orange-50 to-pink-100` с динамическими блендинг-эффектами `mix-blend-multiply` для суб-кругов.

## 2. Элементы интерфейса (Glassmorphism & High Contrast)
*   **Карточки (Glass Canvas):** Белые полупрозрачные контейнеры с эффектом размытия заднего плана.
    *   Стиль: `backdrop-blur-md bg-white/70 border border-white/40 shadow-lg rounded-2xl`
*   **Главные кнопки:** Мягкие пастельные градиенты со сглаженными углами, приятными отбрасываемыми тенями и отзывчивой motion-анимацией масштаба при наведении и нажатии.
*   **Инпуты (Input Fields):** Белые размытые поля с тонкой границей, плавно расширяющиеся и подсвечивающиеся при фокусе.
*   **Текст:** Основной текст — глубокий графитовый/антрацитовый (`text-slate-800`), заголовки с полужирным начертанием (`font-sans font-semibold tracking-tight`).

## 3. Навигация и адаптивность
*   **Десктопная версия (PC Layout):** Верхняя шапка с элегантным брендингом и профилем рядом с боковым меню или удобной сеткой подразделов.
*   **Мобильная версия (Mobile App):** Специальная нижняя панель навигации (Bottom Navbar) со сглаженными кнопками для мгновенного переключения разделов большим пальцем (Touch Targets >= 44px).

## 4. Разделы (Routes / Views)
*   `/start` — Интерактивный онбординг, подключение ботов в 3 клика (Telegram, VK, OK, Дзен, Сетка, etc.).
*   `/channels` — Список подключенных каналов и тарифов (индикатор лимитов: 3 бесплатно, далее 100-200 руб.).
*   `/posts` — Создание, ИИ-рерайт, планировщик по расписанию и генератор постов с цветными кнопками.
*   `/market` — Биржа объявлений между владельцами каналов и размещение в общей премиум ленте.
*   `/bundles` — Модуль взаимного продвижения (подборки/папки) с комиссионным приемом оплат и управлением участниками.
*   `/profile` — Профиль пользователя, баланс токенов, статистика по UTM, кликам, переходам и статистика заработка.
